# Research notes

Long-running investigations that are **not** full product specs: empirical black-box characterization, open hypotheses, and test matrices. When a finding stabilizes, it should graduate into:

- [`docs/guides/scryfall-comparison.md`](../guides/scryfall-comparison.md) (known divergences, operational comparison), and/or
- A [`docs/specs/`](../specs/) spec and implementation, or
- An ADR if it changes parity or product stance (see ADR-019).
