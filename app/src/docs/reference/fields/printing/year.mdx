# year

Release year of the printing.

**Canonical:** `year`  
**Aliases:** —  
**Domain:** Printing

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Release year | `q=year:2021` |
| `=` | Release year | `q=year=2021` |
| `!=` | Not year | `q=year!=2020` |
| `>`, `<`, `>=`, `<=` | Year comparison | `q=year>=2020` |

## Allowed values

Release year (YYYY) or partial match. `q=year:2021` matches printings released in 2021.
