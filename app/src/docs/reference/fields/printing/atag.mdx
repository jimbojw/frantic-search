# atag

Community-tagged artwork from the Scryfall tagger project.

**Canonical:** `atag`  
**Aliases:** `art`  
**Domain:** Printing

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Boundary-aligned prefix match on tag-normalized keys (union) | `q=atag:bolt` |
| `=` | Exact match on the full tag-normalized key | `q=atag=bolt` |
| `!=` | Negates exact `=` only (Frantic extension) | `q=atag!=dragon` |

## Matching

Tag values use **tag normalization** (accents folded, lowercase, **hyphens preserved**; other punctuation stripped)—the same rules as [otag](?doc=reference/fields/card/otag). Hyphens mark **word boundaries** in hyphenated slugs.

- **Prefix (`:`):** **Boundary-aligned prefix** union over illustration tag keys (same semantics as `otag:`).
- **Exact (`=`):** Only keys whose **full** tag-normalized form **equals** the query match.
- **`!=`:** Matches printings that do **not** carry the exact tag from `=`.
- **Empty value:** `atag:` or `atag=` with nothing after the operator matches every **printing** that has **at least one** illustration tag in the loaded index (narrower than empty `kw:`; see [Scryfall differences](?doc=reference/scryfall/differences)).
- **Empty `!=`:** Inverts that set—printings with **no** illustration tags in the loaded index.
- **No match:** Non-empty value with no key matching under the active operator → **`unknown illustration tag "…"`** with query passthrough.

Frantic uses **`:`** / **`=`** / **`!=`** as above; Scryfall targets an **exact** illustration tag label for the filter. **Canonical Scryfall links** may still use unique-prefix resolution. See [otag](?doc=reference/fields/card/otag) and [set](?doc=reference/fields/printing/set).

## Notes

Illustration tags are tied to **printings** (art can differ by edition). See [tag-filter](?doc=reference/special/tag-filter) for `#tag` filtering.
