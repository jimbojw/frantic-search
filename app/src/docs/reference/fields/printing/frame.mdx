# frame

**Canonical:** `frame`  
**Aliases:** —  
**Domain:** Printing

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Frame version | `q=frame:2015` |
| `=` | Frame version | `q=frame=2015` |
| `!=` | Not frame version | `q=frame!=1993` |

## Allowed values

Frame version: `1993`, `1997`, `2003`, `2015`, `future`. Matches the frame style of the printing.
