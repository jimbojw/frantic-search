# collectornumber

Collector number within the set.

**Canonical:** `collectornumber`  
**Aliases:** `cn`, `number`  
**Domain:** Printing

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Normalized **prefix** on stored collector strings (Spec 182) | `q=cn:261` |
| `=` | Normalized **exact** match | `q=collectornumber=261` |

Comparison operators and **`!=`** are not supported (use **`-`** / **`NOT`** to exclude).

## Allowed values

Collector number as printed on the card (numeric or alphanumeric, e.g. `261`, `261a`, `1b`). Matching uses the same **`normalizeForResolution`** rules as other categorical fields on both the query and each printing’s collector string.

A **non-empty** value that matches **no** printing under the active operator returns **`unknown collector number`** (passthrough per Spec 039). **Empty** **`cn:`** / **`cn=`** after trim does **not** narrow results (ADR-022).
