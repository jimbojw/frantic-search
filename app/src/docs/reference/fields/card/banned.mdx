# banned

Cards specifically excluded from a format's legal list (not cards that were never legal, e.g. silver-bordered).

**Canonical:** `banned`  
**Aliases:** —  
**Domain:** Card

## Operators

Same **`:`** / **`=`** / **`!=`** split as [legal](?doc=reference/fields/card/legal), evaluated on the **banned** legality column: **`:`** prefix union over format names, **`=`** exact, **`!=`** negates the exact **`=`** mask only (Frantic extension). Empty **`:`** / **`=`** / **`!=`** are neutral while typing.

| Operator | Example |
|----------|---------|
| `:` | `q=banned:legacy` |
| `=` | `q=banned:modern` |
| `!=` | `q=banned!=legacy` |

## Allowed values

Format codes: `standard`, `modern`, `legacy`, `vintage`, `commander`, `pauper`, etc. Matches cards **banned** in that format.

## Notes

- For format legality, use [legal](?doc=reference/fields/card/legal).
- For restricted (Vintage), use [restricted](?doc=reference/fields/card/restricted).
- Negate a prefix with **`-banned:…`**, not **`!=`**, unless you intend exact-mask negation.
