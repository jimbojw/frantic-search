# restricted

Cards restricted to one copy in a format (e.g. Vintage).

**Canonical:** `restricted`  
**Aliases:** —  
**Domain:** Face (oracle-level)

## Operators

Same **`:`** / **`=`** / **`!=`** split as [legal](?doc=reference/fields/card/legal), on the **restricted** column. Empty **`:`** / **`=`** / **`!=`** are neutral while typing.

| Operator | Example |
|----------|---------|
| `:` | `q=restricted:vintage` |
| `=` | `q=restricted:vintage` |
| `!=` | `q=restricted!=vintage` |

## Allowed values

Format codes. Primarily Vintage; other formats rarely use restricted lists.

## Notes

- For format legality, use [legal](?doc=reference/fields/card/legal).
- For banned lists, use [banned](?doc=reference/fields/card/banned).
- Negate a prefix with **`-restricted:…`** unless you intend exact-mask **`!=`**.
