# salt

EDHREC saltiness (how often players complain about the card).

**Canonical:** `salt`  
**Aliases:** `edhrecsalt`, `saltiness`  
**Domain:** Card

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Saltiness or percentile | `q=salt:2` |
| `=` | Exact saltiness | `q=salt=3` |
| `!=` | Not saltiness | `q=salt!=0` |
| `=null` | No salt score | `q=salt=null` |
| `!=null` | Has salt score | `q=salt!=null` |
| `>`, `<`, `>=`, `<=` | Saltiness comparison | `q=salt>2` |
| `>N%`, `<N%`, etc. | Percentile (saltiness) | `q=salt>90%` |

## Allowed values

- **Numeric:** EDHREC saltiness score. Higher = saltier. `q=salt>2` matches cards with saltiness above 2.
- **Percentile:** Integer 1–99. `q=salt>90%` = top 10% saltiest cards. `q=salt<10%` = bottom 10%.

## Notes

- **`null`:** Use `q=salt=null` to find cards without a salt score. `q=salt!=null` finds cards with salt data. Comparison operators with `null` are invalid.
- Saltiness reflects how often players vote "salt" on EDHREC.
- Percentile is Frantic Search–exclusive (Scryfall does not support it).
