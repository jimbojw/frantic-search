# banned

**Canonical:** `banned`  
**Aliases:** —  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Banned in format | `q=banned:legacy` |
| `=` | Banned in format | `q=banned:modern` |

## Allowed values

Format codes: `standard`, `modern`, `legacy`, `vintage`, `commander`, `pauper`, etc. Matches cards banned in that format.

## Notes

- For format legality, use [legal](?doc=reference/fields/face/legal).
- For restricted (Vintage), use [restricted](?doc=reference/fields/face/restricted).
