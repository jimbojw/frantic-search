# is

Mechanics, layouts, roles, and finish (e.g. commander, dfc, foil).

**Canonical:** `is`  
**Aliases:** `not`  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Has property | `q=is:commander` |
| `=` | Has property | `q=is:dfc` |

## Convenience: not:

The `not:` field is a Scryfall convenience for `-is:`. It uses the same keywords and operators.

| Query | Equivalent to |
|-------|---------------|
| `q=not:dfc` | `q=-is:dfc` |
| `q=-not:dfc` | `q=is:dfc` |

## Allowed values

Keywords for mechanics, layouts, roles, and finish. Examples:

| Category | Examples |
|----------|----------|
| Mechanics | `q=is:commander`, `q=is:legendary`, `q=is:vanilla` |
| Layouts | `q=is:dfc`, `q=is:meld`, `q=is:split` |
| Roles | `q=is:booster`, `q=is:promo` |
| Finish | `q=is:foil`, `q=is:etched`, `q=is:nonfoil` |
| Frame | `q=is:default`, `q=is:atypical` |
| Land cycles | `q=is:fetchland`, `q=is:shockland`, `q=is:pathway` |
| Promo / printing (when printings load) | `q=is:alchemy`, `q=is:rebalanced`, `q=is:rainbowfoil`, … | See below |

Matching is case-insensitive. See the [Syntax cheat sheet](?doc=reference/syntax) for the full list.

## `is:alchemy` (printing-level)

`is:alchemy` matches printings where Scryfall bulk data marks the card as Alchemy: either **`promo_types` includes `alchemy`** or the printing’s **`set_type` is `alchemy`** (e.g. Alchemy: Outlaws of Thunder Junction). Frantic Search follows that data rather than hiding printings that scryfall.com search omits.

For a small edge case, **seven** [Mystery Booster 2](?q=set%3Amb2+unique%3Aprints+include%3Aextras) playtest printings carry the alchemy promo tag in bulk but do not appear in Scryfall’s default search for `is:alchemy`. Frantic includes them. To see exactly those printings, run **[is:alchemy set:mb2](?q=is%3Aalchemy%20set%3Amb2)**.

## Notes

- `q=is:commander` matches cards that can be your commander (legendary creatures, planeswalkers with "can be your commander", etc.).
- `q=is:default` matches printings with the default Magic frame; `q=is:atypical` matches printings with atypical frame treatments (borderless, extended art, showcase, etc.).
- Land cycle keywords (e.g. `q=is:fetchland`) match curated card-name sets.

## Scryfall

Frantic Search supports a subset of Scryfall's `q=is:` keywords. Some values (e.g. `q=new:rarity` for "first printed at new rarity") are not yet implemented. See [Scryfall Differences](?doc=reference/scryfall/differences) for known gaps.
