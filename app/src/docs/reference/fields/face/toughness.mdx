# toughness

Toughness stat for creatures and vehicles.

**Canonical:** `toughness`  
**Aliases:** `tou`  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Equals or has value | `q=tou:5` |
| `=` | Equals | `q=toughness=5` |
| `!=` | Not equal | `q=tou!=1` |
| `=null` | No toughness | `q=tou=null` |
| `!=null` | Has toughness | `q=tou!=null` |
| `>`, `<`, `>=`, `<=` | Numeric comparison | `q=tou>5` |

## Allowed values

Numeric toughness. Use `q=tou:*` to match cards with variable toughness. Non-creatures and cards without toughness are excluded from numeric comparisons.

## Notes

- **`null`:** Use `q=tou=null` to find cards without toughness. `q=tou!=null` finds cards with toughness. Comparison operators with `null` are invalid.
- Negation: `q=-tou>5` ⟺ `q=tou<=5` (operator inversion).
