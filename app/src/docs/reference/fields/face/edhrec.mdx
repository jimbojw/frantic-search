# edhrec

EDHREC Commander popularity rank.

**Canonical:** `edhrec`  
**Aliases:** `edhrecrank`  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Rank or percentile | `q=edhrec:100` |
| `=` | Exact rank | `q=edhrec=50` |
| `!=` | Not rank | `q=edhrec!=1` |
| `=null` | No EDHREC rank | `q=edhrec=null` |
| `!=null` | Has EDHREC rank | `q=edhrec!=null` |
| `>`, `<`, `>=`, `<=` | Rank comparison | `q=edhrec<100` |
| `>N%`, `<N%`, etc. | Percentile (popularity) | `q=edhrec>90%` |

## Allowed values

- **Numeric:** EDHREC Commander popularity rank. Lower = more popular. `q=edhrec<100` matches top 100 most popular.
- **Percentile:** Integer 1–99. `q=edhrec>90%` = top 10% most popular (rank inversion: higher % = more popular). `q=edhrec<10%` = bottom 10%.

## Notes

- **`null`:** Use `q=edhrec=null` to find cards without EDHREC rank. `q=edhrec!=null` finds ranked cards. Comparison operators with `null` are invalid.
- Rank is from EDHREC's Commander data.
- Percentile is Frantic Search–exclusive (Scryfall does not support it).
