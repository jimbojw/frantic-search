# mana value

Converted mana cost / mana value.

**Canonical:** `mana value`  
**Aliases:** `mv`, `cmc`  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Equals or has value | `q=mv:2` |
| `=` | Equals | `q=cmc=2` |
| `!=` | Not equal | `q=mv!=0` |
| `>`, `<`, `>=`, `<=` | Numeric comparison | `q=mv<=2` |

## Allowed values

Numeric mana value (converted mana cost). Hybrid and Phyrexian mana count as the higher cost (e.g. `{b/p}` = 1). `{X}` counts as 0 for MV.
