# restricted

Cards restricted to one copy in a format (e.g. Vintage).

**Canonical:** `restricted`  
**Aliases:** —  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Restricted in format | `q=restricted:vintage` |
| `=` | Restricted in format | `q=restricted:vintage` |

## Allowed values

Format codes. Primarily `q=vintage` for Vintage format; other formats rarely use restricted lists.
