# loyalty

**Canonical:** `loyalty`  
**Aliases:** `loy`  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Equals or has value | `q=loy:3` |
| `=` | Equals | `q=loyalty=3` |
| `!=` | Not equal | `q=loy!=0` |
| `>`, `<`, `>=`, `<=` | Numeric comparison | `q=loy>=3` |

## Allowed values

Numeric loyalty (planeswalker starting loyalty). Only planeswalkers have loyalty; other card types are excluded from numeric comparisons.
