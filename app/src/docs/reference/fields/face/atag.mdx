# atag

Community-tagged artwork from the Scryfall tagger project.

**Canonical:** `atag`  
**Aliases:** —  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Has tag | `q=atag:bolt` |
| `=` | Has tag | `q=atag:bolt` |
| `!=` | Does not have tag | `q=atag!=dragon` |

## Allowed values

Illustration tags are community-curated labels for artwork (e.g. bolt, dragon). Matching is substring-based. Tags come from the tag data model; see [tag-filter](?doc=reference/special/tag-filter) for `#tag` filtering.
