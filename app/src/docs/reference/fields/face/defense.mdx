# defense

**Canonical:** `defense`  
**Aliases:** `def`  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Equals or has value | `q=def:4` |
| `=` | Equals | `q=defense=4` |
| `!=` | Not equal | `q=def!=0` |
| `>`, `<`, `>=`, `<=` | Numeric comparison | `q=def>3` |

## Allowed values

Numeric defense (battle defense, e.g. Siege cards). Only battles have defense; other card types are excluded from numeric comparisons.
