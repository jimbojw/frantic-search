# power

Power stat for creatures and vehicles.

**Canonical:** `power`  
**Aliases:** `pow`  
**Domain:** Face

## Operators

| Operator | Behavior | Example |
|----------|----------|---------|
| `:` | Equals or has value | `q=pow:4` |
| `=` | Equals | `q=power=4` |
| `!=` | Not equal | `q=pow!=0` |
| `=null` | No power (non-creatures) | `q=pow=null` |
| `!=null` | Has power | `q=pow!=null` |
| `>`, `<`, `>=`, `<=` | Numeric comparison | `q=pow>=4` |

## Allowed values

Numeric power. Use `q=pow:*` to match cards with variable power (e.g. "X", "*"). Non-creatures have no power and are excluded from numeric comparisons.

## Notes

- **`null`:** Use `q=pow=null` to find non-creatures (e.g. Lightning Bolt). `q=pow!=null` finds creatures. Comparison operators with `null` are invalid.
