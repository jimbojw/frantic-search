# Fields Overview

Filter fields are grouped by **what part of the game object** they describe:

| Group | Meaning | Examples |
|-------|---------|----------|
| **Card** | Oracle-wide: same for every face and (usually) every printing of that card | `q=ci:w`, `q=otag:ramp`, `q=produces:w` |
| **Face** | This **face** on a card: name, type line, mana cost, colors on this side, P/T on this face, etc. | `q=name:bolt`, `q=t:creature`, `q=c:green` |
| **Printing** | This **edition** / physical printing: set, price, frame, illustration tags, etc. | `q=set:mh2`, `q=usd<5`, `q=atag:bolt` |

`is:` mixes **face** and **printing** keywords; see the [is](?doc=reference/fields/face/is) article for which is which.

When you combine fields from different groups, results are still **printing-level** when any printing condition is involved: each row is a specific printing that matches. Use `q=unique:cards` (or `q=++` for unique prints) to control deduplication.

## Operators

| Operator | Meaning | Example |
|----------|---------|---------|
| `:` | Contains / has at least | `q=o:destroy` |
| `=` | Exactly equals | `q=c=rg` |
| `!=` | Not equal | `q=c!=r` |
| `>` | Greater than | `q=pow>3` |
| `<` | Less than | `q=mv<3` |
| `>=` | Greater or equal | `q=tou>=5` |
| `<=` | Less or equal | `q=cmc<=2` |

Not all fields support all operators. For example, `q=game:` supports only `:`, `=`, and `!=`. See each field's article for details.

## Card fields

- [banned](?doc=reference/fields/card/banned), [legal](?doc=reference/fields/card/legal), [restricted](?doc=reference/fields/card/restricted) — Format legality (stored per printing; usually uniform across printings of an oracle)
- [edhrec](?doc=reference/fields/card/edhrec), [salt](?doc=reference/fields/card/salt) — EDHREC popularity and saltiness
- [identity](?doc=reference/fields/card/identity) — Color identity
- [my](?doc=reference/fields/card/my) — Cards in a list
- [otag](?doc=reference/fields/card/otag) — Oracle (function) tags
- [produces](?doc=reference/fields/card/produces) — Mana the card can produce

## Face fields

- [name](?doc=reference/fields/face/name) — Card name (substring, range, percentile)
- [oracle](?doc=reference/fields/face/oracle) — Rules text
- [type](?doc=reference/fields/face/type) — Type line
- [color](?doc=reference/fields/face/color) — Colors on this face
- [power](?doc=reference/fields/face/power), [toughness](?doc=reference/fields/face/toughness), [loyalty](?doc=reference/fields/face/loyalty), [defense](?doc=reference/fields/face/defense) — Oracle P/T, loyalty, defense: numeric, substring, or exact match depending on operator and value shape (see each article)
- [mana value](?doc=reference/fields/face/mana-value), [mana](?doc=reference/fields/face/mana) — Converted mana cost and mana cost symbols
- [is](?doc=reference/fields/face/is) — Layouts, oracle mechanics, roles, and printing traits (see article)
- [kw](?doc=reference/fields/face/kw) — Keyword ability

## Printing fields

- [artist](?doc=reference/fields/printing/artist) — Artist name (substring)
- [atag](?doc=reference/fields/printing/atag) — Illustration (art) tags
- [collectornumber](?doc=reference/fields/printing/collectornumber) — Collector number
- [date](?doc=reference/fields/printing/date) — Release date
- [flavor](?doc=reference/fields/printing/flavor) — Flavor text (substring or regex)
- [frame](?doc=reference/fields/printing/frame) — Frame version
- [game](?doc=reference/fields/printing/game) — Paper, Arena, MTGO availability
- [in](?doc=reference/fields/printing/in) — Game, set, or rarity (disambiguated by value)
- [rarity](?doc=reference/fields/printing/rarity) — Rarity
- [set](?doc=reference/fields/printing/set) — Set code
- [set_type](?doc=reference/fields/printing/set_type) — Scryfall set type (e.g. expansion, memorabilia)
- [usd](?doc=reference/fields/printing/usd) — Price in USD
- [year](?doc=reference/fields/printing/year) — Release year
