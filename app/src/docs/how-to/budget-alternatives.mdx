# Find Budget Alternatives

1. Search for the card you want to replace (e.g. `n:rhystic`).
2. Add a price filter: `usd<5` or `usd=null` for unpriced.
3. Use `otag:` or `atag:` to find similar cards by community tags.
4. Sort by `usd` to see cheapest printings first.
